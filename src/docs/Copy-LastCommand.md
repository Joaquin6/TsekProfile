# Copy-LastCommand

## SYNOPSIS
Copies the last command to the clipboard.

## SYNTAX

```
Copy-LastCommand
```

## DESCRIPTION
Leverages the Get-History cmdlet to identify the last command
executed and copies it to the clipboard by leveraging the Set-Clipboard cmdlet.

## EXAMPLES

### EXAMPLE 1
```
Copy-LastCommand
```
